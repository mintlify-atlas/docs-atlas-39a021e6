## Introduction

Justina Core Backend integrates an advanced AI analysis engine that evaluates surgical simulations in real-time. The AI system connects to the backend via WebSocket to receive notifications of completed surgeries, retrieves trajectory data, performs multi-step analysis, and returns actionable feedback to surgeons.

## Architecture

The AI system operates as a separate Python service that communicates with the Spring Boot backend:

```
┌─────────────────┐         WebSocket          ┌──────────────────┐
│                 │◄──────────────────────────►│                  │
│  Spring Boot    │     NEW_SURGERY events     │  Python AI       │
│  Backend        │                            │  Engine          │
│                 │                            │                  │
└────────┬────────┘                            └────────┬─────────┘
         │                                              │
         │  GET /surgeries/{id}/trajectory             │
         │◄────────────────────────────────────────────┤
         │                                              │
         │  POST /surgeries/{id}/analysis              │
         │◄────────────────────────────────────────────┤
         │                                              │
```

## Data Flow

1. **Surgery Completion**: Surgeon finishes simulation, backend finalizes surgery session
2. **WebSocket Notification**: Backend sends `NEW_SURGERY` event to connected AI clients
3. **Trajectory Retrieval**: AI fetches movement data via GET endpoint
4. **Analysis Execution**: 5-step pipeline processes trajectory and generates metrics
5. **Result Submission**: AI posts score and feedback via POST endpoint
6. **Persistence**: Backend saves analysis results to database

## Authentication

The AI system authenticates as a special user with `ROLE_AI` permissions:

<CodeGroup>
```python config.py
import os

BASE_URL = os.getenv("BACKEND_URL", "http://localhost:8080")
IA_USERNAME = os.getenv("IA_USERNAME", "ia_justina")
IA_PASSWORD = os.getenv("IA_PASSWORD", "ia_secret_2024")
```

```python Authentication Flow
class JustinaAIClient:
```

## Security Permissions

The backend restricts AI endpoints to authenticated AI users:

```java SecurityConfig.java
.requestMatchers(HttpMethod.GET, "/api/v1/surgeries/*/trajectory")
    .hasAnyAuthority("ROLE_SURGEON", "ROLE_AI")
.requestMatchers(HttpMethod.POST, "/api/v1/surgeries/*/analysis")
    .hasAuthority("ROLE_AI")
```
